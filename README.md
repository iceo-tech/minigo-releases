# Minigo — AI Desktop Assistant

Download Minigo to install and use OpenClaw AI agent with a friendly GUI.

## Downloads

| Platform | Architecture | File |
|----------|-------------|------|
| macOS | Apple Silicon (M1/M2/M3/M4) | `Minigo-1.0.12-arm64.dmg` |
| macOS | Intel | `Minigo-1.0.12.dmg` |
| Windows | x64 | `Minigo-Setup-1.0.10.exe` |

## Installation

### macOS
1. Download the `.dmg` file for your Mac
2. Open the DMG and drag **Minigo** to Applications
3. First launch: Right-click → Open (required for unsigned apps)
4. Follow the setup wizard inside the app

## System Requirements

- **macOS**: 10.15 (Catalina) or later, 4GB RAM, 500MB disk
- **Windows**: 10/11 64-bit, 4GB RAM, 500MB disk

## Website

Enable GitHub Pages on this repo (Settings → Pages → Source: main branch) to host the download page at:

```
https://YOUR-USERNAME.github.io/REPO-NAME/
```

## Auto-Update

The `latest-mac.yml` file enables auto-update for the Electron app. Future versions will be detected automatically when users open Minigo.
